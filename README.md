## вљ пёЏ Troubleshooting

| Problem | Fix |
|---------|-----|
| `ModuleNotFoundError: timm` | `pip install timm` |
| `CUDA out of memory` | Reduce `BATCH = 32` in `train_model.py` |
| Weather API timeout | Check internet; script falls back to synthetic data |
| Models not found in app | Run `python train_model.py` first |
| Streamlit port in use | `streamlit run app.py --server.port 8502` |
| Slow CPU training | Reduce `CNN_EP=5`, `MM_EP=10` for quick test |
